# Employee Management System Project

A modern, resonsive web application for managing employee records efficiently.


# Technologies Used

- HTML5 
- CSS3 
- Vanilla JavaScript 

# Installation

1. Clone the repository 
2. Open index.html file in your browser

## Features

- Dashboard 
- Employee Management 
- Add Employee 
- Edit Employee 
- Delete Employee 
- Search and Filter 
- Pagination 
- Responsive Design 

# Uses
- Use the **Login** page for sign with the credentials `admin `/`password`.
- Use the **Dashboard** page for dashboard
- Use the **Employees** page for employee management
- Use the **Add Employee** page for adding new employee
